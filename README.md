# Config Auditor

Configuration file security auditor.

Auditor keamanan file konfigurasi.


## Description

Scans configuration files for hardcoded secrets and insecure settings.

## Deskripsi

Memindai file konfigurasi untuk secret hardcoded dan pengaturan tidak aman.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Audit: python config_audit.py /path/to/config
- Demo: python config_audit.py --demo

## Penggunaan

- Audit: python config_audit.py /path/to/config
- Demo: python config_audit.py --demo
